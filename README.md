# 📊 Sales & Profit Dashboard (Excel)

## 📌 Overview
This project is an interactive Excel dashboard built to analyze sales and profit data.

## 🚀 Features
- Sales by Category
- Profit by Year
- Customer Insights
- Monthly Trends
- Interactive Slicers (Category, Year, Month)

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Visualization
